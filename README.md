# Prompt

vue + ts + vite로 한가위 웹페이지용 콘텐츠 좀 만들어줘(app.vue)  
각 섹션은 버튼 눌러야 내용이 나오는 거라고 생각하고 만들어줘  

섹션 내용은 이렇게
1. '한가위의 의미' - 2~3문장, 가족이랑 풍요 느낌 강조
2. '전통 음식' - 송편, 한과, 전 이런 거 소개, 2~3문장
3. '전통 놀이' - 윷놀이, 제기차기 이런 거 소개, 2~3문장
4. '한가위 스토리' - 달 관련 전설이나 재밌는 이야기, 2~3문장

각 content는 버튼 누르면 나오는 텍스트라고 생각하고 작성하고,  
전체적으로 따뜻하고 감성적으로, 가족끼리 보는 느낌 나게
